# How to set up the voice assistant framework for you AI Agent

This is a super quick tutorial on how to set up the voice assistant framework for your AI Agent.

### The one and only step:

All you need to do is just run the `Voice_assistant.py` file and it will automatically install everything needed. It uses Jarvis as the wake word. You can change the wake word, refer to https://github.com/dscripka/openWakeWord for more info on how to do that. Put your new model (if you want one) in the models folder and change the model name in the `Voice_assistant.py` file. You can also change the TTS and STT models in the same way.


### How to use the voice assistant:
You will have to say Jarvis, then it will print heard Jarvis when it hears the wake word. After that say your question, which it will also print out, then it prints out the AI Agent's response, and also say it out loud!
